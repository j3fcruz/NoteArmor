## рЯІ† Usage Guide

2. Create new tabs with `Ctrl+N`
3. Write your notes freely across multiple tabs
4. Save encrypted: `Ctrl+Shift+S` вЖТ choose **Encrypted** вЖТ set password

6. To open: `Ctrl+O` вЖТ select `.notearmor` вЖТ enter password to decrypt
